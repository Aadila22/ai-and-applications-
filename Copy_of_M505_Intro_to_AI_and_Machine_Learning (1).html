Machine Learning - Based Customer Segmentation for Mall Marketing Analysis¶

Business Content and Introduction¶

Shopping malls attract a wide range of customers, each with unique preferences, spending habits, and purchasing power. In today's competitive retail environment, it is important for mall management and retail businesses to understand these differences in order to provide personalized shopping experiences and increase profits.

Customer segmentation is key to addressing this issue. By using data-driven methods customers characteristics like age, income, and spendings in which mall authorities can gain valuable insights into consumer behavior. These insights can help marketing teams create targeted campaigns, loyalty programs, and promotions that appeal to each customer segment.

To tackle this challenge, a machine learning model can be created to segment customers smartly and automatically using clustering algorithms like K-Means.

Problems - Machine Learning¶

The business question lends itself neatly to a classic classification pipeline in machine learning. Our aim is to predict a price-range category that reflects different levels of customer spending: 0 Budget-friendly, 1 Mid-range, 2 Premium, and 3 Luxury.

To tackle the problem well, we will evaluate multiple machine learning methods so far learned.

The Dataset used for analysis was publicly obatained from kaggle :https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

def classify_price_range(income):
    if income <= 30:
        return 0  # Budget Friendly
    elif income <= 60:
        return 1  # Mid Range
    elif income <= 90:
        return 2  # Premium Range
    else:
        return 3  # Luxury Range

mall_customers_df['price_range'] = mall_customers_df['Annual Income (k$)'].apply(classify_price_range)

Limitations¶

  • The dataset is quite small, so the model might miss some patterns and may not work perfectly on new customers.

  • Some customer groups have fewer examples than others, which can make the model better at predicting common groups and less accurate for smaller ones.

  • We only used simple information like age, income, and spending — but other factors might also affect customer behavior that we didn’t include.

  • The model assumes a straight-line relationship between features and price range, so it might not catch more complicated patterns.

Conclusion¶

So, in this project, I used Logistic Regression to split mall customers are categorized based on age, income, and spending habits. Model got about 86% accuracy, which was pretty cool — it shows these features really help us understand customer behavior.

This model can help mall managers and shops figure out who their customers are and target them better with personalized deals and marketing. That way, they can boost sales and keep shoppers happy.

In the future, I’d love to try more advanced methods like combining different models or even deep learning to make it even smarter. Also, adding things like real-time shopping trends could help the model adjust to changes in customer behavior.
